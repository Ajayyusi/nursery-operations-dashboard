# Nursery Operations Dashboard

## Project Overview
This is a business intelligence dashboard project for nursery operations. It focuses on enrollment, payments, attendance, inquiries, expenses, and management reporting.

## Business Problem
Nursery management often needs to monitor student enrollment, payment collection, attendance, inquiries, and operational costs across different classes and months. Manual tracking in spreadsheets can make reporting slow and limit visibility into business performance.

## Solution
Built a dashboard-ready BI project using anonymized sample data. The project includes structured datasets, SQL analysis, DAX measures, and a dashboard screenshot for portfolio presentation.

## Tools Used
